Hurricane Sandy Twitter Response

In recent years, the world has seen an increase in the frequency and intensity of natural disasters resulting from climate change. In these scenarios, social media platforms like Twitter can serve as a free and open distributed sensor network. If relevant information can be identified from the stream of Tweets, we can obtain local and timely information well before the news travels through traditional communication channels. This project analyzes Twitter streams using Natural Language Processing and Machine Learning techniques to map out power outages in the areas affected by Hurricane Sandy in real time.

The Data

The dataset contains over 4.5 million geo-tagged Tweets sent from Washington DC and 13 affected states within 12 days of Hurricane Sandy. The Tweets were scraped using Twitter's API. The data was in JSON format with a total size of about 15 GB. The relevant information such as Tweet text, geolocation, and timestamps etc. was extracted and analyzed using Python Pandas and postprocessed data was saved in CSV files.

The figure below shows number of Tweets per hour as a function of time. The Hurricane Sandy related Tweets were filtered out using key words such as Sandy, Hurricane, Storm, etc. The Sandy related Twitter activity peaks near the landfall, Oct. 29, 2012.

Hashtag statistics

The hastags #S(s)andy, #hurricaneS(s)andy, #frankestrom, #hurricane are more common around the landfall. We also see other popular hashtags reflecting other significant events happening around the same time, for example #debates, #halloween.

County Level Choropleth Maps

The county level choropleth map below shows geographical distribution of all Sandy related Tweets, power outage related Tweets, and Sentiment. The power outage related Tweets were filtered using a Naive Bayes classifier trained using a small subset containing about 500 Tweets heuristically (manually) determined to be relevant to power outage. All of the 4.8 million Tweets were labeled using this clasifier, of which 7,000 were found to be positive examples.
It is important to take into account the significant variation in the number of Tweets originating from different zip code areas, densly populated areas tend generate more Tweets. To take this into account, we compare statistics as normalized by the total number of Tweets originating from each county. Not surprisingly, power outage and sentiment seem to be negatively correlated.

Spatiotemporal Maps

The map below shows location of power outages as a function of time. There are virtually no power outages until Oct. 28, before the landfall. The number of power outages shoots up on Oct. 29. Most of the power outages seem to persist for several days after the landfall.
